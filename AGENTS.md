# AGENTS.md

## Project overview

`slopkick` is a Pi coding-agent extension that adds the `/slopkick` terminal review UI. It lets users review diffs, annotate lines/files/whole changes, and insert a follow-up prompt back into Pi.

The package is published to npm as `slopkick` and is loaded by Pi from `./src/index.ts`.

## Git guidance

Use Conventional Commits for commit messages, for example `feat: add shortcut customization`, `fix: handle empty diffs`, or `docs: update install instructions`.
